# Hello MNIST in 4 Frameworks

Simple MNIST classifier written in PyTorch, PyTorch Lightning, TensorFlow 2,
and Keras.

## Install Dependencies

```bash
pip install -r requirements.txt
```

## PyTorch / Lightning

```bash
# pytorch
python pytorch.py

# lightning
python pytorch_ligtning_mnist.py
```

## Tensorflow/keras

```bash
# keras
python keras.py

# tensorflow
python tf2.py
```
